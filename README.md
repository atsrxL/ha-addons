# atsrxL Home Assistant Add-ons

Custom Home Assistant add-ons maintained for personal smart home services.

## Add-ons

- [Wyoming Sherpa ASR](./wyoming_sherpa_asr): Fast Chinese speech-to-text using sherpa-onnx Paraformer on port `10303`. Approx memory: `0.5-1 GB`.

## Installation

Add this repository to Home Assistant:

```text
https://github.com/atsrxL/ha-addons
```
